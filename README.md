# Simple Crush Prank 💘😂

A fun and lightweight prank web app that generates fake “crush compatibility” results.
Built using simple HTML, CSS, JavaScript, and Flask backend support.

Perfect for learning web development, UI design, and creating funny projects for friends.

---

## 🚀 Features

* Fake crush percentage generator
* Smooth and simple UI
* Random funny results
* Stores prank logs
* Responsive design
* Easy to run locally

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Ravibagda462006/simple-cush-prank.git
```

Go to project folder:

```bash
cd simple-cush-prank
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
python app.py
```
